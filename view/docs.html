üì• Media Downloader

Download audio and media files from various sources

Blue Archive Voice

GET /api/downloader/bluearchive-voice

Parameters

character optional

Character name (e.g., Hina, Aru, Yuuka, Shiroko). If not specified, returns random character voice.

Example Request

GET /api/downloader/bluearchive-voice?character=Hina

Example Response

{
  "status": true,
  "query": "Hina",
  "title": "Hina_BattleVoice_Victory_01.ogg",
  "audio_url": "https://static.wikitide.net/...",
  "character": "Hina",
  "total_found": 47
}

Sound Effects (MyInstants)

GET /api/downloader/myinstants

Parameters

query required

Search term for sound effects (e.g., "bruh", "wow", "victory")

Example Request

GET /api/downloader/myinstants?query=bruh

Example Response

{
  "status": true,
  "query": "bruh",
  "data": [
    {
      "id": "bruh-sound-12345",
      "title": "Bruh Sound Effect",
      "url": "https://www.myinstants.com/instant/bruh/",
      "mp3": "https://www.myinstants.com/media/sounds/bruh.mp3"
    }
  ],
  "total_found": 15
}

YouTube Video (MP4)

GET /api/downloader/youtube-mp4

url required

YouTube video URL (supports youtube.com and youtu.be)

Example Request

Example Response

{
  "status": true,
  "creator": "Nanan",
  "result": {
    "title": "Video Title",
    "result": "https://tinyurl.com/...",
    "quality": "720p",
    "size": "25.3 MB",
    "thumb": "https://i.ytimg.com/...",
    "views": "1.2M",
    "channel": "Channel Name",
    "uploadDate": "2024-01-01",
    "desc": "Video description..."
  }
}

üó£Ô∏è Text-to-Speech

Convert text to speech audio in multiple languages

Sound of Text

GET /api/soundoftext

Parameters

text required

Text to convert to speech

lang optional

Language code (e.g., "en-US", "id-ID", "ja-JP"). Default: "en-US"

Example Request

GET /api/soundoftext?text=Hello World&lang=en-US

Example Response

{
  "status": true,
  "text": "Hello World",
  "lang": "en-US",
  "audio_url": "https://example.com/audio.mp3"
}
